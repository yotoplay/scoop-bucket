# Scoop Bucket for twine-to-yoto

[Scoop](https://scoop.sh) bucket for [twine-to-yoto](https://github.com/yotoplay/twine-to-yoto).

## Usage

```powershell
scoop bucket add yotoplay https://github.com/yotoplay/scoop-bucket
scoop install twine2yoto
```

## Updating

```powershell
scoop update twine2yoto
```
